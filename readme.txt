== Description ==

The Recent Posts With Featured Image Widget adds a widget that displays the 3 most recent posts, with featured image, within any widget area. Responsive design and code allows this widget to work well with all themes. 

== Installation ==

1. Upload `/recent-posts/` to the `/wp-content/plugins/` directory or upload the recent_postsJM.zip file using the add new plugin feature within the WordPress admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'Recent Posts with Image' widget to your desired widget area.
